MERIS FR L1B v2013

The recently reprocessed dataset of MERIS full resolution (v2013) is available as input. The dataset has been geo-corrected with AMORGOS 4.0 and is used as MERIS FSG for processing.

A further quality check of this dataset has been requested and is currently performed by ESA and the IDEAS team to blacklist files that have obvious issues. The current v2013 dataset still contains products with issues that had been identified and removed in the previous dataset.

2013-10-21

SPOT VGT P data available for LC-CCI

The complete archive of SPOT VGT P data is on the cluster. We got it from VITO for the LC CCI project. The global dataset spans the period from 1998 to 2013.

2013-09-20

Point data format check

There is a new button "Check" below the point data selection field in the MA workflow tab. This button checks the selected file for syntax and reports what it has recognised. It can be applied without side effects, the data remains unchanged. Files that do not pass the check will usually also fail during processing.

2013-08-09

Catalogue

Calvalus input data is being registered in a new MERCI instance available to internal (!) users at

http://calvalus-merci:8080/merci

The full mission MERIS FSG, MERIS RRG, and one year (2003) of AATSR have been registered so far. Users can log into MERCI with their Calvalus credentials.

NB. The access to quicklooks and metadata details is currently not very stable because of bugs in the Hadoop fuse implementation.

Hardware resource monitoring

Ganglia has been configured for the cluster available to users at

http://www.brockmann-consult.de/ganglia/
